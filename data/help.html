Summary

This addon is an academic prototype called TasksTodo. It allows you to manage your goals and tasks at hand and to save and restore relevant web pages during the different web sessions. It also offers different task-related functions like showing past web search queries or query suggestions. That way it will hopefully support you at your task-related information search activities on the web.

Background

Projects with a complex goal like writing a funding proposal or learning a new and complex topic can take hours, days, or even months to complete. In such cases, information gathering requires more than just a single search session. To achieve such a goal, it is necessary to specify, execute, and complete derived tasks, since they define concretely what has to be done. During the different task stages, the knowledge requirements vary and change from the more general to the more specific. Both, the stage and the complexity of a task influence how we seek and search for information.

With the support of TasksTodo, we enable users to structure their goals at hand into tasks and sub-tasks without leaving the browser or switching the application. Implicit (e.g., executed queries or visited web pages) and explicit (e.g., bookmarks) interactions are stored with reference to the current task in order to offer an organized task-based history that eases task resumption. Additionally, this data is used to build higher-level support features, like the adaption and personalization of the retrieved web content, or to provide modified support functionalities that match the task-related relevance. To help users better recapture past activities, TasksTodo shows past web search queries. Moreover, it also shows suggestions regarding the current search query.

TasksTodo allows users to easily create an arbitrary number of goals. Within each goal any number of tasks can be defined and structured on different levels (i.e., a task that contains sub-tasks). The integrated activity tracking, is implemented as an extensive logging mechanism that records all user operations and assigns this data to the current task. This log data contains the use of the prototype itself (i.e., how goals and tasks are managed), as well as the activities done in the browser (e.g., page visits or executed search queries). For instance, a page visit is logged in the form of a screenshot, the page title, the URL, and the content. Search activities are tracked, including the information about the search provider and the query. This way all behavioral data is stored implicitly in relation to the current goal and task, which allows us to present a task-based browsing history that provides an overview of past task-based activities in order to re-capture and continue interrupted tasks more easily.

Data Privacy

Data privacy is crucial. Therefore, data is stored in the local profile directory of your browser and is in control of the user. If the user delete entries from the task-related history, these data is physically deleted from the database. Form inputs like credentials are not stored. To offer support services like suggestions, data is used and anonymously sent to third party providers.

Navigation

Navigation is mainly done by clicking the defined goals and tasks. The top navigation allows you to easily switch between goals or to open this help page.

Goals

Goals are an important concept of TasksTodo, because they are the abstract definition of what has to be achieved. Examples of a goal are writing a proposal for a converence, learning a new and complex topic (e.g., learning a programming language), or the necessarity of making decisions like buying a new car. In TasksTodo the user can define an arbitary number of goals within each any number of different tasks can be added that define what has to be done to achieve that goal.

Create a new goal

You can create a new goal by clicking on the button "New goal". This opens a small text editor which allows you to define the title of your new goal. Store it by clicking on "Save".

Modify a goal

To modify the name of the goal, click on the pencil symbol (), which appears on the right side after hovering over the goal title with the mouse pointer. This operation can not be undone.

Delete a goal

To modify the name of the goal, click on the trash symbol (), which appears on the right side after hovering over the goal title with the mouse pointer. This operation can not be undone.

Mark a goal as completed

To mark a goal as completed, click on the checkbox symbol on the left. The goal title gets lined through and is written in italic letters. This operation can be undone.

Tasks

Tasks are also an important concept of TasksTodo, because they concretly define what has to be done to achieve a goal. Users can add an arbitary number tasks within a certain goal.

Create a new task

You can create a new task by clicking on the button "New task". This opens a small text editor which allows you to define the title of your new task. Store it by clicking on "Save".

Modify a task

To modify the name of the task, click on the pencil symbol (), which appears on the right side after hovering over the task title with the mouse pointer. This operation can not be undone.

Delete a task

To modify the name of the task, click on the trash symbol (), which appears on the right side after hovering over the task title with the mouse pointer. This operation can not be undone.

Select a task

All browsing and searching activities are stored to the selected task. Therefore, it is necessary that the user selects a task by clicking on the task title. After task selection, the user will get a selection notification and the task options () are shown.

Mark a task as completed

To mark a task as completed, click on the checkbox symbol on the left. The task title gets lined through and is written in italic letters. This operation can be undone.

Latest search queries

Depending on your task the application shows your task related queries on Google, Bing, Yahoo, Google Scholar, YouTube and Wikipedia.

Notes

Personal notes

This option lets you take personal notes to a task. Simply select your task an select the notes icon to open the notes window. Click on the "new" button to quickly add your notes.

Note by selection

There is another way to take notes by selecting text on any website and add the selection as a note to your task.

Bookmarks

Bookmark current page

You can easily bookmark web pages to your current task (precondition is the selection of a task). Just open the context menu of the browser on the web page (not on a link) and select the option "Add this page as a bookmark to my current task" as shown in the screenshot to the right.

Bookmark linked page

You can easily store a bookmark by simply clicking on a link to open the browsers' context menu. Precondition is the selection of a certain task. Than select the option "Add this link as a bookmark to my current task". After this a new bookmark appears in the bookmark list of the current task.

History

TasksTodo stores the browsing history when you work on your tasks. Therefore it is necessary that you select a task in your task list. TasksTodo than automatically saves all visited web pages to the tasks' history. This can be helpful when working on longitudinal tasks, which last over different sessions. In these situations it allows you to easily recap your last session(s). You can access the history clicking on the history icon of a selected task.

Attachments

Attach files by upload

You can easily add files as an attachment to your tasks. Each task has its own file repository where you can add files to. To do so you have to select a task and click on the attachments icon. Then simply select your files on your local machine and drag them to the drag point. The upload begins and the files will appear in the attachments list.

Attach files by link

Another way to add attachments to your task is to save linked files. To do so open the context menu of the browser on a link, which links to the file as shown in the example in the left. This calls a function on the server to download this file for you and to store it as an attachment to your task.
